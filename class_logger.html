Detailed Description

Allows csv logging.

Can be run on a constant frequency in its own thread, or on a on-demand base. First declare a logger, then add the SharedData variables you want to log. Constant frequency logging: run StartPeriodicLog(frequency) Log on-demand: call LogHeader() and LogLine()
