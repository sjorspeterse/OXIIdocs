Detailed Description

Basic functionality that applies to all Nuna components.

Every Nuna component has two responsibilities: formatting and processing. Formatting could for example be extracting all values of the steering wheel CAN frame into properly named variables. Processing could for example be the RadarSensor tracking a vehicle. A Sensor is also a Subscriber of Buffers, and a Notifier of Actuators.
