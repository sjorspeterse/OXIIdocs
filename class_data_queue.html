template<typename T>
bool DataQueue< T >::pop ( T &  result)
inline

Pops item from queue and stores it in result.

If the data is expired, it clears all data, sets data object to default and returns false. This makes sure that for example when steer sends gas command and is disconnected, after the expire time it sets the gas value to 0.
