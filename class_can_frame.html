void CanFrame::SwapBytes ( )

Swaps the order of the bytes in the CAN frame.

Byte 0 becomes byte 7, byte 1 becomes byte 6, etc. This is useful when the datasheet (e.g. Mitsuba motorcontroller) specifies the last byte as byte 0.
